FUNNY WORLD is a browser game hub made with HTML, CSS, and Vanilla JavaScript. It includes action games, pixel puzzle games, branching story games, and a funny shopping game where you spend Elon Money.



## Pages

| Page | Description |
| --- | --- |
| Home | Main landing page with navigation to all game categories. |
| Game | Fast arcade games: Endless Runner and Ball Catch. |
| Pixel Game | Puzzle games: Sudoku, Memory Match, Pixel Puzzle, and Tic Tac Toe. |
| Story Game | Interactive story choices with adventure and horror themes. |
| Elon Money | Shopping game where you spend a huge virtual budget. |

## Games Included

### Game Tab

- **Endless Runner** - jump, duck, avoid obstacles, and keep your score growing.
- **Ball Catch Game** - move the paddle, catch falling balls, and survive as the speed increases.

### Pixel Game Tab

- **Sudoku** - fill the 9x9 board and check your answer.
- **Memory Match** - flip cards, find matching pairs, and beat your move count.
- **Pixel Puzzle** - swap puzzle pieces until the image is complete.
- **Tic Tac Toe** - play the classic 3x3 strategy game.

### Story Game Tab

- **Adventure stories** - choose your path and reach different endings.
- **Horror stories** - make decisions in suspenseful scenes with story images.

### Elon Money Tab

- Start with **$200,000,000,000**.
- Buy luxury items from the shop.
- Track purchased items in the shopping cart.
- See the total spent and remaining money update instantly.

## Features

- Multi-page website structure

## Run Locally

No installation is required.

1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Use the navigation bar to play each game category.
